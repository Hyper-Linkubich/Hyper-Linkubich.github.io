## Skapa olika ikoner
Oftas räcker det med en favicon.ico på 32x32 px och en ikon med storleken 512x512 px.  
Blir dock bättre med flera png-ikoner.  

## Font-family

```css
@font-face {
    font-family: 'digitalDigits';
    src: url('/font/SUBWT___.ttf') format('truetype');
}
```
